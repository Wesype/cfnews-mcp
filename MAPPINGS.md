## Ordre de Tri (sort_type)

- Croissant: `ascending`
- Décroissant: `descending`

## Notes d'Usage

1. **Encodage URL**: Tous les paramètres doivent être encodés en URL
2. **Arrays**: Les paramètres multiples utilisent la notation `param[]=value`
3. **Pagination**: Utiliser `?page=X` pour naviguer entre les pages
4. **Limit**: Utiliser `&limit=X` pour récupérer plus de résultats (max 10x la taille de page)

## Exemples de Requêtes Complètes

### Opérations LBO Biotech 2024

```
op_type[]=271&sector[]=124&depuis=01/01/2024&jusquau=31/12/2024&sort_attribute=fiche_operation_operation_date_value_dt&sort_type=descending
```

### Fonds VC en levée

```
vehicle_segment[]=189608&vehicle_status[]=189636&sort_attribute=fiche_vehicule_operation_date_value_dt&sort_type=descending
```

### Sociétés Fintech IDF

```
sector[]=305&uniqut_istech=oui&soc_region[]=132336&sort_attribute=attr_ca_f&sort_type=descending
```
